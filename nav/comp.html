Computation Formulas

This page provides information on the formulas used for GPA calculations.

SGPA (Semester Grade Point Average) Calculation:

The SGPA is calculated using the following formula:

SGPA = Œ£(Creditsi * Gradei) / Œ£Creditsi

Where:

  • Creditsi: Credits of the ith course.
  • Gradei: Grade obtained in the ith course (on a scale of 0 to 10).

CGPA (Cumulative Grade Point Average) Calculation:

The CGPA is calculated by taking the weighted average of all the SGPA values up to the current semester:

CGPA = Œ£(SGPAi * Creditsi) / Œ£Creditsi

Where:

  • SGPAi: SGPA of the ith semester.
  • Creditsi: Total credits completed up to the ith semester.

Conversion of SGPA to CGPA:

To convert SGPA to CGPA, you can use the following formula:

CGPA = (Œ£(SGPAi * Creditsi) + SGPAcurrent * Creditscurrent) / (Œ£Creditsi + Creditscurrent)

Where:

  • SGPAi: SGPA of the ith semester.
  • Creditsi: Total credits completed up to the (i-1)th semester.
  • SGPAcurrent: SGPA of the current semester.
  • Creditscurrent: Credits of the current semester.

Feel free to use the provided formulas in your GPA calculations!
